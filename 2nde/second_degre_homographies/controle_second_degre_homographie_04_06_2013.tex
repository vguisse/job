\begin{exercice}
 La\baremeexo{7} fonction $f$ est définie sur $\R$ par :
\[
 f(x)=3x^2-3x-6.
\]
\begin{enumerate}
 \item Donner\baremeque{2} en justifiant le tableau de variations de $f$. Préciser la valeur des 
éventuels maximum ou minimum de $f$.
 \item Prouver\baremeque{1} que pour tout nombre réel $x$ :
\[
 f(x)=3(x+1)(x-2).
\]
  \item Prouver\baremeque{1} que pour tout nombre réel $x$ :
\[
 f(x)=3[(x-\frac12)^2-\frac54].
\]
  \item Quelle est l'image de\baremeque{1} 0 par $f$?
  \item Quels sont\baremeque{1} les antécédents de 0? Faire figurer cette information dans le tableau de variations de $f$.
  \item Résoudre\baremeque{1} $f(x)=-6$.
\end{enumerate}
\end{exercice}
